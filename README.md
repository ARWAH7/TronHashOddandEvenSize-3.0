# 波场 (TRON) 哈希实时分析大盘 3.0 专业版

这是一个专为波场区块链设计的专业级哈希数据实时分析与可视化平台。系统通过高频对接 TronGrid API，实现对区块哈希末位数字（单双、大小）的深度统计、趋势绘图及智能长龙提醒。

## 🚀 3.0 版本核心进化

在 2.0 稳定版基础上，3.0 版本引入了多项工业级增强特性：

### 1. 智能双模长龙提醒系统 (Dual-Engine Dragon Alerts)
系统现在支持两种维度的长龙监控，并采用左右并排的直观布局：
- **走势序列长龙**：实时监控各采样步长下的最新连续出球情况（垂直时间序列）。
- **珠盘行级长龙**：独家支持对珠盘路（Bead Road）进行**横向行级分析**。系统会扫描珠盘网格的每一行，当某一物理行在连续列中出现相同结果时触发预警。

### 2. 高效批量管理面板
- **批量规则编辑**：支持通过 CSV 格式文本一键导入/导出数十条采样规则（名称、步长、偏移、网格行数、龙阈值）。
- **一键阈值同步**：支持快速将选定的长龙提醒阈值（如 5 连、8 连）应用到全局所有规则，极大提升配置效率。

### 3. 深度可视化矩阵
- **多维度盘面**：集成综合大盘、单双走势、大小走势、单双珠盘、大小珠盘五大视图。
- **自定义网格系统**：每条采样规则均可独立配置走势图（大路）与珠盘图的行数（3-60行），适配不同分析习惯。

## 🛠 功能详情

### 📊 专业路图分析
- **大路 (Trend Chart)**：采用经典“换列”逻辑，直观展示连出与跳跃趋势。
- **珠盘路 (Bead Road)**：按区块生成顺序精确填充，支持物理行级长龙扫描。
- **当前统计**：每个图表头部实时显示该维度下的当前长龙状态及历史总计数。

### ⚙️ 核心采样逻辑
- **步长切换**：支持从 1 区块到 1000 区块的任意采样频率。
- **起始偏移**：通过设置起始区块，可实现特定“组”或“对子”的精准对齐。
- **断线重连**：内置自动轮询机制，智能处理网络波动，确保数据流不间断。

### 🎨 极致 UI/UX 体验
- **动态主题配色**：支持自定义单、双、大、小的全局 UI 颜色。
- **响应式架构**：完美适配 PC 宽屏显示与移动端快速查阅。
- **实时同步状态**：底部悬浮条实时显示 API 连接状态及区块捕获进度。

## 📦 技术实现方案

- **Frontend**: React 19 + TypeScript (强类型定义确保数据计算准确)。
- **Styling**: Tailwind CSS + Framer Motion (实现丝滑的动画过渡)。
- **API Support**: 直接集成 TronGrid V1 协议，支持 Pro-API-Key 鉴权。
- **Storage**: 基于浏览器 LocalStorage 实现配置的持久化存储。
- **Algorithm**:
  - `calculateTrendGrid`: 动态递归算法，处理路图折叠逻辑。
  - `HorizontalRowScanner`: 针对珠盘路的行级跨列扫描算法。

## 🚦 快速开始

1. **配置 API Key**：点击右上角“设置”，输入您的 TronGrid API Key（必填）。
2. **定义规则**：使用批量编辑或手动新增，设定您关心的区块采样步长。
3. **监控长龙**：在“长龙提醒”选项卡中，关注实时弹出的预警卡片，系统会自动计算预期中奖的区块高度。
