# 波场 (TRON) 哈希实时分析大盘 3.0 专业版

本平台是一款专为波场区块链（TRON Network）爱好者和专业分析师打造的实时数据可视化分析工具。通过直接对接波场主网 API，系统能够秒级捕捉最新的区块哈希数据，并基于哈希末位的数字特性（单双、大小）提供全方位的路图分析与智能预警。

---

## 💎 3.0 版本：重塑分析体验

在 3.0 版本中，我们引入了行业领先的“双模长龙预警引擎”和“极速批量配置”功能，使数据监控的深度与效率达到了新的高度。

### 1. 智能双模长龙提醒 (Dual-Engine Alerts)
系统将监控维度拆分为两个核心板块，支持左右并排实时对比：
- **走势序列长龙 (Trend Streak)**：监控特定采样步长下的最新垂直产出序列。发现最直观的单双、大小连出规律。
- **珠盘行级长龙 (Bead Row Streak)**：**业内首创**。针对珠盘路（Bead Road）网格进行横向物理行扫描。当某一物理行在连续列中出现相同结果时触发。

### 2. 工业级批量管理系统
- **CSV 批量编辑**：支持通过文本框一键导入/导出所有采样规则。
- **全局阈值同步**：支持一键将“长龙提醒阈值”应用到所有规则。

## 📊 核心可视化矩阵

### 专业路图分析
- **标准大路图 (Trend Chart)**：采用经典“换列”逻辑，精准展示连出（向下生长）与跳跃（向右换列）的趋势。
- **物理珠盘路 (Bead Road)**：严格按区块产出顺序填充。支持自定义 3-60 行网格高度。

### 实时统计数据
- 每个图表模块均内置实时计数器，自动统计当前窗口内的：
  - 单/双、大/小 总数比例。
  - 当前正在进行的连出状态（长龙统计）。

## 🛠 采样逻辑说明

- **动态步长 (Interval)**：支持 1 到 5000 区块的任意间隔采样。
- **起始偏移 (Offset)**：允许设置特定起始高度，实现精准的“组对”分析。
- **断线自动回补**：系统内置智能轮询与缓存机制，确保数据流不间断。

## 🎨 设计与交互亮点

- **极致配色自定义**：支持针对“单、双、大、小”四个状态自定义全局 UI 颜色。
- **响应式布局**：完美兼容 PC 宽屏、平板以及手机移动端。
- **全状态浮动条**：页面底部实时显示 API 通讯状态、区块同步进度。

## 📖 使用指南

本工具为纯前端应用，配置信息存储于您的浏览器本地：

1. **配置 API**：点击右上角“设置”图标，填入您的 TronGrid API Key。
2. **设定规则**：手动新增或使用“批量编辑”导入规则。
3. **开始监控**：在“综合盘面”查看全局走势，或切换至“长龙提醒”专注捕获预警。
