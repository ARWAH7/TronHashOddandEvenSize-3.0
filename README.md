
# 波场 (TRON) 哈希实时分析大盘 3.6 AI 增强版

本平台是一款专为波场区块链（TRON Network）深度分析师打造的实时数据可视化大盘。3.6 版本在 AI 预测、实时同步与用户体验上实现了全方位突破，为哈希趋势研究提供军工级的技术支持。

---

## 🚀 3.6 版本核心进化

### 1. 🌊 智能动态滚动系统 (Intelligent Adaptive Scrolling) - **NEW**
彻底解决了长周期观察中的交互痛点：
- **初次加载自适应**：系统首次获取数据或切换采样规则时，自动平滑滚动至最右侧，确保用户第一时间看到最新区块。
- **智能跟随逻辑**：仅当用户视角处于“最新数据区”时开启自动跟随；若用户正在回溯历史，系统将静默更新，不干扰用户的深度分析。

### 2. 📏 自适应高度引擎 (Dynamic Height Engine) - **NEW**
走势图（大路）与珠盘路组件现在完全响应规则参数：
- **行数动态适配**：根据配置的“走势行数”或“珠盘行数”自动调整 UI 容器高度。
- **消除视觉冗余**：彻底移除下方多余空白区域，使大盘布局在移动端与桌面端均达到极致紧凑。

### 3. 🧠 AI 深度演算 4.0 (AI Stable Engine v4.0)
集成最新的马尔可夫链与贝叶斯推演模型，针对 3.6 版本进行了稳定性微调：
- **高确定性过滤**：仅在哈希熵值处于特定低频区间且模型置信度 >92% 时输出信号。
- **共振模型字典**：新增模型逻辑详解，帮助用户理解每一个“单双”与“大小”决策背后的统计学依据。

## 💎 核心功能矩阵

- **综合盘面 (Dashboard)**：
  - 支持单双、大小四路图表同屏监控。
  - 实时统计出现次数、当前长龙状态，并伴有“火热（Flame）”视觉提醒。
  
- **长龙榜精细化筛选器**：
  - 针对 200+ 采样规则，提供单、双、大、小四维过滤，瞬间从海量数据中锁定长龙机会。
  
- **全量规则快速切换器 (Quick Switcher)**：
  - 快捷键级搜索跳转，支持模糊检索采样步长，适配专业分析师的极速切换需求。

## 💻 本地部署 (Local Deployment)

系统采用 React 19 + TypeScript + Tailwind CSS 构建，纯前端驱动，数据安全透明。

1. **安装环境**：
   ```bash
   npm install
   ```

2. **运行项目**：
   ```bash
   npm run dev
   ```

3. **核心配置**：
   - 在设置中输入您的 **TronGrid API Key** 即可开启每秒一次的高频同步模式。

## 📖 深度使用建议

1. **利用右侧行号**：珠盘路新增右侧行号标识，配合“珠盘行长龙”提醒，精准定位每一行的横向规律。
2. **多维关注联动**：在长龙榜中点击“爱心”关注核心趋势，AI 预测引擎会优先针对您的关注项进行深度分析。
3. **颜色自定义**：设置中支持对“单、双、大、小”进行 16 进制颜色微调，打造您的专属视觉仪表盘。
