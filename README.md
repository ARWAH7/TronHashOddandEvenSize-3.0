
# 波场 (TRON) 哈希实时分析大盘 3.0 专业版

本平台是一款专为波场区块链（TRON Network）深度分析师打造的实时数据可视化大盘。通过直接对接波场主网 API，系统能够秒级捕捉最新的区块哈希数据，并基于哈希末位的数字特性（单双、大小）提供工业级的路图分析与智能预警逻辑。

---

## 💎 3.0 旗舰版本：海量数据下的极致管理

在 3.0 版本中，我们重构了交互逻辑，完美解决了在配置 **100-200+ 采样规则** 时的导航与切换痛点。

### 1. 🚀 全量规则快速切换器 (Quick Switcher)
当您拥有 120+ 条规则时，传统的导航栏无法一次性展示所有内容。3.0 引入了**智能切换中心**：
- **一键唤起**：点击导航栏左侧的“网格图标”即可开启全屏检索模态框。
- **毫秒级检索**：支持通过“规则名称”或“采样步长”（如输入 `120`）立即定位并跳转。
- **网格平铺**：所有规则以紧凑网格形式平铺，让您对整套监控体系一目了然。

### 2. 🔥 智能双模长龙提醒 (Dual-Engine Alerts)
系统实时监控各维度的连出情况，并在“长龙提醒”标签页集中展示：
- **走势序列长龙 (Trend Streak)**：监控垂直产出序列。例如：步长为 10 的采样点连续 5 次产出“单”。
- **珠盘行级长龙 (Bead Row Streak)**：**行业首创**。针对珠盘路（Bead Road）网格进行横向物理行扫描。当某一物理行在跨列连续产出相同结果时触发。

### 3. 🛠 工业级批量管理系统
- **CSV 批量导入**：支持 `名称,步长,偏移,走势行,珠盘行,龙阈值` 格式。可在数秒内完成整套 1-1000 步长的规则链条部署。
- **全量阈值同步**：支持一键将“长龙连出提醒”阈值应用到所有活跃规则（例如：统一设置为 8 连即报）。
- **极速检索与排序**：在设置中心，您可以按步长从小到大排序，或模糊搜索快速修改特定规则。

## 💻 本地部署 (Local Deployment)

系统采用 React 19 + TypeScript + Tailwind CSS 构建，具备极致的运行速度与离线存储能力。

1. **安装依赖**：
   ```bash
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **生产环境构建**：
   ```bash
   npm run build
   ```

## 📊 核心可视化矩阵

- **综合大盘 (Dashboard)**：同屏展示单双/大小的走势图（大路）与珠盘图。
- **深度分析 (Deep Dive)**：每个维度均支持全屏独立分析。
- **数据明细 (Data Table)**：实时流式展示原始区块 Hash、高度及解析结果，支持点击跳转 TronScan 浏览器。

## 📖 深度使用技巧：管理 120+ 规则

1. **利用水平滚动**：导航栏支持鼠标滚轮左右滑动或点击箭头快速翻阅。
2. **善用 Quick Switcher**：如果您记得步长，直接点击左侧图标输入数字跳转，这是最快的方式。
3. **批量维护**：定期使用“批量编辑”导出配置作为备份，或在 Excel 中批量调整规则参数后再粘贴回系统。
